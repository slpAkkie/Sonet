# Sonet Frontend

Frontend часть проекта [Sonet](https://github.com/slpAkkie/Sonet)

## Начало работы

```sh
npm install
```

### Компиляция и запуск live-сервера

```sh
npm run serve
```

### Компиляция и минификация для продакшена

```sh
npm run build
```

### Линт и исправление файлов

```sh
npm run lint
```
